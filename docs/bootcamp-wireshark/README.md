# Bootcamp Wireshark

Initieel was het niet de bedoeling om dit onderdeel op te nemen in deze cursus. Maar bij de try-outs cursus werd het snel duidelijk dat er onvoldoende kennis van Wireshark aanwezig was bij de cursisten.  
Dit onderdeel is optioneel naar gelang de aanwezige kennis.  

## Wat is Wireshark?

Wireshark is een *packet sniffer* en *proctocol analyzer*. De kracht van Wireshark ligt voornamelijk in de capaciteiten om protocollen te analyseren en minder in het capteren van netwerkverkeer. Momenteel kan Wireshark meer dan 3000 protocollen dissecteren.
